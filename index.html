Section 2

Semantic Elements Summary

Semantic elements are elements that describes its meaning in a readable way to both the browser and developer. One of the most prominent examples of semantic elements is the HTML 5 programming language. While non-semantic elements are still being commonly used on websites today, semantic elements clearly defines its content. With the trend toward semantic code use, there are a number of benefits to its usage such as:

  1. Easier to Read
  2. Greater Accessibility

Section 4

Text Formatting in HTML Examples

  • Italic Tag - displays text in italics.
  • Bold Tag - displays text in bold.
  • Strong Tag - displays text in bold. Used to make important emphasis.
  • Emphasis Tag - another emphasis tag that displays text in italics.
  • Subscripted Tag - defines subscript text, like the two atoms of oxygen in CO2.
  • Superscripted Tag - defines a superscript like the power of a number, 102.
  • Small Tag - reduces the size of text.
  • Deleted Tag - defines deleted text by striking a line through the text.
  • Inserted Tag - defines inserted text which is usually underlined.
  • Short Quoted Tag - is used for shorter inline quotes.
  • Cited Tag - is used for citing the author of a quote.
  • Code Tag - displays code snippets.
  • Marked Tag - highlights text.
